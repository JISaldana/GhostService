==========================================================
             GHOSTSERVICE - MOUSE UTILITY
==========================================================

What is GhostService?
---------------------
It is a small tool designed to prevent your computer from 
entering sleep mode or your application status from 
switching to "Inactive" while you are away.

How does it work?
-----------------
1. Run "GhostService.exe".
2. A ghost icon will appear in the system tray 
   (next to the Windows clock).
3. The program occasionally interacts with the system to 
   check how long it has been idle.
4. When the system detects a certain period of inactivity, 
   the program will perform circular cursor movements 
   to keep the system active.

How to close it?
----------------
Right-click on the ghost icon in the taskbar and 
select "Close" (or "Cerrar").

Security:
---------
For your safety, this program:
   - Does NOT log keystrokes (No keylogging).
   - Does NOT require administrator privileges.
   - Does NOT connect to the internet.
   - Is an open-source tool focused on privacy.

Developed with C++ and WinAPI.
